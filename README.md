# Skyblock Death Mod
 
A Fabric client-side mod for Hypixel Skyblock Dungeons that automatically sends party chat messages when deaths are detected.
 
## Features
 
- Detects when any party member is killed by a mob and announces it in party chat
- Detects when you die and sends a self-callout in party chat
- Detects tank deaths (mob-related) and calls them out
- Detects trap deaths and calls them out
- Zero performance impact — only reads chat messages, no polling or ticking
